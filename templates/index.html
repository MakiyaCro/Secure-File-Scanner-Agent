[ VULNSCAN — SYSTEM INFORMATION ]

VULNSCAN is a local-only AI-powered code vulnerability scanner.

All analysis runs on your machine via Ollama + Qwen3.5. No code leaves your system.

[ SECURITY MODEL ]

✓ Files scanned for prompt injection before analysis

✓ Path traversal prevention on all file operations

✓ Rate limiting: 10 requests per 60 seconds

✓ 50 MB total upload cap, 500 KB per file cap

✓ 100 file maximum per session

✓ No code execution — static analysis only

✓ Session-isolated file storage

✓ Security headers on all responses

.rb .swift .kt .scala .sh .sql .html .css .json .yaml

.toml .env .cfg .conf .xml .tf .hcl

[ SETUP REQUIREMENTS ]

1. Install Ollama: https://ollama.com

2. Pull model: ollama pull qwen3.5

3. Start Ollama: ollama serve

4. Run VulnScan: python app.py
